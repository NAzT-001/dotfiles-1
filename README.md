# My dotfiles

Configure mac. Setup and management using [rcm](http://thoughtbot.github.io/rcm/rcm.7.html).

## Scripts

### Bootstrap

First setup installation run the up command.

```
$ ./up.sh
```

### Resave

After adding a new dotfile resave the install script.

```
$ dotup
```
